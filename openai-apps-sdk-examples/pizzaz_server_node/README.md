# Pizzaz Carousel MCP server (Node)

This server packages the `pizza-carousel` tool using the official Model Context Protocol (MCP) TypeScript SDK. It serves the inline HTML snippet generated by the root `build-all.mts` script so the Apps SDK can render the carousel beside assistant responses.

## Prerequisites

- Node.js 18+
- pnpm (or your preferred package manager)

## Install dependencies

```bash
pnpm install
```

Run the command inside this directory. It installs only the server dependencies.

## Start the server

```bash
pnpm start
```

The server listens on `http://localhost:8000` and exposes an SSE endpoint at `/mcp`. Ensure you build the widget assets from the repository root (`pnpm run build`) before starting so that `assets/pizzaz-carousel.snippet.html` exists.

## Tool behaviour

- **pizza-carousel**  
  - Requires an input object with a single `pizzaTopping` string.  
  - Returns plain text, echoes the topping through `structuredContent`, and includes `_meta.openai/outputTemplate` metadata pointing to `ui://widget/pizza-carousel.html`.
  - Embeds the inline carousel HTML generated by the build step.

Modify `src/server.ts` to integrate real data or add more tools once you generate additional widget bundles.
