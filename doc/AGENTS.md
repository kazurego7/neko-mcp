# AGENTS.md

このリポジトリでは、OpenAI Apps SDK と MCP を使った実験を進めています。  
ここでは、プロジェクトに関わるエージェントの役割とタスクの整理方針をメモします。

## 役割整理
- **Cat Gallery Agent**
  - CatAPI から猫画像を収集し、Apps SDK のウィジェットに最適化された形式で提供。
  - 画像の権利表記やクレジットを適切に添付する。
- **Conversation Orchestrator**
  - ユーザーと AI の会話の流れを管理し、猫要素を挟み込むトリガーを決める。
  - ChatGPT 側のトーンガイドラインに従い、通知や再登場のタイミングを制御する。
- **UX Designer Agent**
  - インラインカルーセルやオーバーレイなど UI のスタイルガイドを策定。
  - ガイドラインの更新を監視し、必要に応じてコンポーネントを調整する。

## タスク管理ポリシー
- 最初はスピードを優先し、動作するプロトタイプを素早く揃える。
- 細部の改善やアクセシビリティ強化は、気づいたタイミングで順次反映。
- ガイドラインの変更や CatAPI の制約については定期的にレビューする。
